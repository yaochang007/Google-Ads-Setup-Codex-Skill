# Google Ads Performance Max Draft Setup Checklist

## Preflight

- [ ] Account name and customer ID were confirmed.
- [ ] Approved Performance Max plan was available.
- [ ] Gate B approval was present.
- [ ] Conversion tracking readiness was confirmed.
- [ ] Budget, bidding, final URL expansion, URL exclusions, brand exclusions,
      asset groups, assets, and audience signals matched the approved plan.
- [ ] Merchant Center/feed readiness was confirmed if ecommerce applies.

## Safety

- [ ] Setup stopped before final save.
- [ ] Setup stopped before publish, enable, or launch.
- [ ] No billing, account access, conversion action, Merchant Center link,
      recommendation, enabled campaign budget, or enabled campaign bidding
      change was made.
- [ ] Any unexpected modal or mismatch caused a stop.

## Handoff

- [ ] Draft summary was produced.
- [ ] Final launch review was identified as the next step.
